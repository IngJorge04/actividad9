Bienvenido a Actividad 9

Un menú interactivo con spotlight, calculadora funcional con historial, documentación en acordeón y galería multimedia.

✨ Glassmorphism 🎨 Gradientes dinámicos ⚡ Interacciones fluidas

Características principales

  • Menú overlay con efecto spotlight siguiendo el cursor
  • Calculadora con soporte de teclado y operaciones encadenadas
  • Historial de cálculos persistente con opción de limpieza
  • Documentación expandible con acordeón
  • Galería multimedia responsive
  • Sistema de login con sesión persistente
  • Perfil de estudiante con datos personales

    Explicación detallada de la estructura HTML, estilos CSS y lógica JavaScript.

    Estructura HTML ▼ 
    <header> con topbar fijo
<main> con secciones de navegación
  - #login: formulario de autenticación
  - #inicio: hero con presentación
  - #calculadora: interfaz de cálculo
  - #codigo: documentación
  - #galeria: grid multimedia
  - #datos-personales: perfil del estudiante
<footer> con información adicional
    Estilos CSS ▼ 
    Variables CSS custom properties
Glassmorphism con backdrop-filter
Gradientes radiales y lineales
Animaciones keyframes para orbes
Transiciones suaves en hover
Grid layouts responsivos
Media queries para móvil
